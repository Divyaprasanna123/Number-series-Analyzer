# Number Series Pattern Analyzer

A Java-based program that analyzes a given number series and detects mathematical patterns.  
The program identifies the pattern used in the sequence and predicts the next numbers in the series.

## Features

- Detects **Arithmetic Series**
- Detects **Multiplication Pattern**
- Detects **Increasing Difference Pattern**
- Detects **Square Number Series**
- Detects **Fibonacci Series**
- Predicts the **next number**
- Predicts the **next three numbers**
- Displays a **visual representation of the series**

## Technologies Used

- Java
- Arrays
- Loops
- Conditional Statements
- Scanner Class

## How It Works

1. The user enters the size of the number series.
2. The user inputs the numbers.
3. The program calculates the differences between elements.
4. It detects the pattern in the series.
5. The program predicts the next values.
